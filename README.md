## 📱 Overview

**Medical App** is a React Native mobile application tailored for healthcare professionals. It serves as a centralized hub for medical news, educational topics, clinical tests, and professional development resources. Built with a modern architecture, it delivers a seamless experience across iOS and Android.

---

## ✨ Key Features

- **Medical News Feed** – Stay current with the latest healthcare developments.
- **Educational Topics** – Browse and learn from curated medical content.
- **Clinical Tests** – Assess your knowledge with interactive quizzes.
- **Rich Content Display** – HTML rendering with WebView support for complex medical articles.
- **Offline Capabilities** – Data persistence using AsyncStorage.
- **Real‑time Updates** – Powered by React Query for efficient server‑state management.
- **Analytics & Monitoring** – Sentry for error tracking, App Center for analytics and crash reports.
- **Responsive Design** – Optimized for all screen sizes, from iPhone SE to 12 Pro Max.
